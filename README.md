* Animated typing effect
* Clean terminal interface
* Motivational output

##  How to Run Projects

1. Clone the repository:

```bash
git clone https://github.com/your-username/interactive-python-greeter.git
```

2. Navigate to a project folder:

```bash
cd day-01-greeter
```

3. Run the project:

```bash
python main.py
```
